# AP Exam FRQs

This source covers AP Calculus BC free-response exams and scoring guides. The
OCR/LaTeX pipeline lives in `../../tools/ocr-frq`.

## Canonical Inputs

- Exam PDFs: `../../tools/ocr-frq/PDFs/BC-*.pdf`
- Scoring-guide PDFs: `../../tools/ocr-frq/PDFs/SG-BC-*.pdf`
- Mathpix exam exports: `../../tools/ocr-frq/mathpix/BC-*.zip`
- Mathpix scoring-guide exports: `../../tools/ocr-frq/mathpix/SG-BC-*.zip`

## Current Outputs

- Combined LaTeX: `outputs/combined-latex/output_frq.tex`
- Combined PDF: `outputs/combined-latex/output_frq.pdf`
- V2 review outputs: `outputs/v2-review`

## Tracking Files

- `CATALOG.md` tracks exam coverage and extracted question count.
- `QUALITY.md` tracks known OCR/scoring-guide limitations.
- `outputs/` exposes generated LaTeX/PDF and review artifacts.
