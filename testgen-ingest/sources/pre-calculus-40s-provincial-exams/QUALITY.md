# Quality

## Current Assessment

The January 2013 sample has a usable generated JSON and review PDF, but it is
not yet a trusted question bank. Treat it as extracted, not fully catalogued.

## Known Risks

- Mathpix artifacts remain possible in body text, solutions, and rubrics.
- Image placement and image-question association need manual review.
- Curriculum classification exists in the converter but has not been audited against Manitoba standards.
- Point values come from marking-guide parsing and should be checked before import.

## Ready Criteria

- Every question has a stable source exam, question number, and point value.
- Body, solution, and images match the original PDF.
- Math renders correctly in the target format.
- Curriculum unit/section assignment has been reviewed.
