# ExamView By Curriculum

These folders index the top-level curriculum families in the ExamView BNK corpus.
They are navigation and review folders, not duplicate BNK storage.

The canonical BNK files remain in `../../../tools/bnk-decoder/ignore/ExamView/Banks`
so existing decoder commands still work from the decoder worktree.

Use `../CATALOG.md` for file counts and `../QUALITY.md` for readiness rules.
